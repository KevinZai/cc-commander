# CC Commander — Product Hunt Listing Draft

## Tagline (60 chars max)
AI project manager for Claude Code — 358 skills, one install

## Description (260 chars max)
CC Commander wraps 16 best-in-class Claude Code packages into one install with an Intelligence Layer that scores tasks, recommends skills, and learns from every session. Arrow-key menus. No commands to memorize. Works in CLI, Claude Desktop, and VS Code.

## Key Features
1. **Intelligence Layer** — Scores tasks 0-100, auto-tunes model/turns/budget
2. **358 Skills** — Deduplicated from 1,500+ across 16 vendor packages
3. **Zero Typing** — Choose-your-own-adventure menus for everything
4. **Knowledge Compounding** — Every session makes the next one smarter
5. **Works Everywhere** — CLI, slash commands, Cowork plugin, VS Code, headless API

## Maker Comment Draft
"I built CC Commander because stock Claude Code is a blank terminal. No skills. No guidance. No memory.

After 14 months and 200+ community articles, I distilled the best patterns into one install — with an Intelligence Layer that scores tasks, recommends skills, and actually learns from your sessions.

It's MIT licensed (Commons Clause on the intelligence core). Try it: `npm install -g cc-commander && ccc`"
