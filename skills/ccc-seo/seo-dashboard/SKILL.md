# SEO Dashboard

Generate a comprehensive SEO health report combining data from all specialist areas.

## Dashboard Sections

### 1. Technical Health Score (0-100)
- Core Web Vitals status (LCP, FID, CLS)
- Mobile usability
- Index coverage (pages indexed vs submitted)
- Crawl errors
- HTTPS status
- Structured data validity
- Sitemap health

### 2. Content Performance
- Top 10 performing pages (by traffic)
- Content freshness (% updated in last 90 days)
- Thin content pages (<300 words)
- Duplicate content issues
- Missing meta descriptions / titles
- Content gap analysis vs competitors

### 3. Backlink Profile
- Total referring domains
- Domain authority / rating
- New vs lost links (trend)
- Toxic link percentage
- Top linking domains
- Anchor text distribution

### 4. Ranking Performance
- Keywords in top 3 / top 10 / top 100
- Ranking trend (improving / declining / stable)
- Featured snippet opportunities
- SERP feature presence

### 5. AI Search Readiness
- robots.txt AI crawler policy
- Structured data coverage
- Content citability score (are answers extractable?)
- Markdown twin availability

## Output Format
Generate a structured markdown report with scores, trends, and prioritized action items. Group actions by effort (quick wins / medium / major).
