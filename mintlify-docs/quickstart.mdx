## Pick Your Path

One question: **How are you using Claude?**

<Tabs>
  <Tab title="Path A: CLI (Recommended)">
    You use `claude` in your terminal. This is the most common way.

    ```bash
    npm install -g cc-commander
    ```

    Then launch the interactive CLI:

    ```bash
    ccc
    ```

    **What you get:** Full CLI with tmux split mode, daemon, theme switching, cockpit dashboard, and all 455+ skills.

    **Best for:** Developers who live in the terminal.
  </Tab>

  <Tab title="Path B: Slash Commands Only">
    You don't want a separate CLI binary. Just skills + commands inside Claude Code sessions.

    ```bash
    curl -fsSL https://raw.githubusercontent.com/KevinZai/cc-commander/main/install-remote.sh | bash
    ```

    Then in any Claude Code session:

    ```
    /ccc
    ```

    Full interactive menu appears. Same features, no extra binary.

    **What you get:** All 455+ skills and commands accessible via `/ccc` prefix.

    **Best for:** Developers who prefer staying inside Claude Code.
  </Tab>

  <Tab title="Path C: Claude Desktop">
    You use Claude Desktop and want CC Commander as a Cowork plugin.

    ```
    /plugin marketplace add KevinZai/cc-commander
    ```

    Say "start commander" or "what should I work on" to begin.

    **What you get:** 7 Cowork skills — project management, infrastructure, knowledge base, and night mode.

    **Best for:** Claude Desktop users who want guided AI workflows.
  </Tab>
</Tabs>

## Your First 60 Seconds

```
1. ccc → Main menu (14 options)
2. Pick "Build something new" → Choose project type
3. Answer 3 multiple-choice questions → CCC generates a plan
4. CCC dispatches Claude with the right model, budget, and skills
5. Session ends → CCC extracts lessons → gets smarter next time
```

No YAML. No API keys. No configuration.

## Skill Management

After install, manage your skill tiers with the built-in browser:

```bash
ccc --skills
```

```
  Skill Management
  ─────────────────────────────────
  Installed: 37 essential skills
  Available: 455 total

  ❯ list         → show installed skills
    install      → add a skill by name
    remove       → remove a skill
    tier         → switch tier (essential/recommended/full)
    search       → find skills by keyword
    browse       → browse all 455+ skills
```

## Verify Your Install

```bash
ccc --test
```

Runs 22-point self-verification. All green means you're ready.

```bash
ccc --status
```

Health check JSON — shows gateway, skill count, session state.
