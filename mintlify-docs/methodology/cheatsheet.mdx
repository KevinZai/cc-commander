## Split Mode (tmux)

```bash
ccc --split        # Launch tabbed tmux mode
Ctrl+A n           # Next tab
Ctrl+A p           # Previous tab
Ctrl+A 0           # Back to CCC menu tab
Ctrl+A q           # Quit session
```
