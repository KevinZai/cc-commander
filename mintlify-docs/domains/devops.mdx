**Production Deployment Pipeline:**
1. `cicd-pipeline-generator` → build pipeline with tests and staging
2. `zero-downtime-deploy` → blue-green or canary deploy strategy
3. `performance-monitoring` → APM and alerting setup
4. `infra-runbook` → incident response playbook

**Container Security Hardening:**
1. `docker-development` → optimized, minimal base images
2. `container-security` → vulnerability scanning, runtime policies
3. `aws-iam-security` → least-privilege container roles
4. `github-actions-security` → supply chain defense in CI
