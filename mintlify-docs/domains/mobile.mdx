## Framework Selection Guide

| Need | Framework | Why |
|------|-----------|-----|
| Cross-platform, JS team | React Native | Largest ecosystem, Expo simplifies setup |
| Cross-platform, performance-critical | Flutter | Near-native performance, single codebase |
| iOS-only, modern UI | SwiftUI | Apple's current standard, great tooling |
| Android-only, modern UI | Jetpack Compose | Google's current standard, Material Design 3 |
