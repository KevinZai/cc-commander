## Pre-Launch Security Review

The recommended flow before any production deployment:

```
secrets-scanner      → find and remove hardcoded credentials
dependency-audit     → check all dependencies for known CVEs
owasp-top-10         → verify against OWASP Top 10 checklist
security-audit       → comprehensive manual review
codeql-integration   → set up automated scanning in CI
```

## AI Application Security

For apps that call LLMs or expose AI features:

```
prompt-injection-defense  → harden all LLM-facing inputs and outputs
secrets-scanner           → verify AI service API keys are not exposed
```

Prompt injection is the most common vulnerability in AI apps — and the one most often skipped in standard security reviews.
