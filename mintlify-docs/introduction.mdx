> **Every Claude Code tool. One install. An AI brain that learns.**

CC Commander is not a skill pack. It's an AI project manager that scores complexity, reads your stack, and gets smarter every session. Built by distilling 200+ community sources into one install.

```bash
npm install -g cc-commander
```

```bash
ccc
```

Arrow keys to navigate. Enter to select. That's it.
